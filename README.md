# Unity Wordle

**Unity Wordle**は、隠された単語を推測するゲームです。
PCとiPhoneでの動作は確認しました。

## 遊び方

1. 隠された単語を推測し、入力フィールドに単語を入力する。
2. 「Enter」キーを押すか、「Submit」ボタンをクリックして、推測を送信する。
3. 推測の結果は以下の通り表示される。
   - **緑色**: 文字が正しい位置にある。
   - **黄色**: 文字は単語に含まれているが、位置が異なる。
   - **灰色**: 文字は単語に含まれていない。
4. 指定された回数以内に正しい単語を推測することが目的である。
